# 全面安全检查清单

> 覆盖主机、网络、应用三层的完整安全检查框架

**适用场景：** 网站部署运维前的安全基线检查

---

## 一、主机安全（Host Security）

### 1.1 系统基础

| 检查项 | 命令 | 标准 |
|--------|------|------|
| 系统版本 | `cat /etc/os-release` | 使用 LTS 版本 |
| 内核版本 | `uname -r` | 最新安全补丁 |
| 系统补丁 | `apt list --upgradable` | 无高危未修复漏洞 |
| 时间同步 | `timedatectl status` | NTP 同步正常 |

### 1.2 用户与权限

| 检查项 | 命令 | 标准 |
|--------|------|------|
| root 登录 | `grep PermitRootLogin /etc/ssh/sshd_config` | no |
| 密码策略 | `cat /etc/pam.d/common-password` | 复杂度要求 |
| sudo 权限 | `cat /etc/sudoers` | 最小权限原则 |
| 空密码账户 | `awk -F: '($2=="")' /etc/shadow` | 无 |
| 最近登录 | `lastlog` | 无可疑登录 |

### 1.3 服务与进程

| 检查项 | 命令 | 标准 |
|--------|------|------|
| 开放端口 | `ss -tlnp` / `netstat -tlnp` | 仅必要端口 |
| 运行服务 | `systemctl list-units --type=service` | 无多余服务 |
| 自启动项 | `systemctl list-unit-files --state=enabled` | 最小化 |
| 异常进程 | `ps auxf` | 无可疑进程 |

### 1.4 日志审计

| 检查项 | 命令 | 标准 |
|--------|------|------|
| 系统日志 | `journalctl -xe` | 正常 |
| 认证日志 | `/var/log/auth.log` | 无暴力破解 |
| 日志轮转 | `ls /etc/logrotate.d/` | 配置正常 |
| 日志保留 | `cat /etc/rsyslog.conf` | ≥90 天 |

### 1.5 文件系统

| 检查项 | 命令 | 标准 |
|--------|------|------|
| 磁盘使用 | `df -h` | <80% |
| 敏感文件权限 | `ls -la /etc/shadow` | 600 |
| SUID 文件 | `find / -perm -4000` | 无异常 |
| 世界可写文件 | `find / -perm -002` | 无 |

## 二、网络安全（Network Security）

### 2.1 防火墙

| 检查项 | 命令 | 标准 |
|--------|------|------|
| 防火墙状态 | `ufw status` / `firewalld-cmd --state` | 开启 |
| 入站规则 | `ufw show added` | 默认拒绝 |
| 出站规则 | `ufw show added` | 有控制 |
| NAT/转发 | `cat /proc/sys/net/ipv4/ip_forward` | 按需开启 |

### 2.2 网络服务

| 检查项 | 命令 | 标准 |
|--------|------|------|
| SSH 端口 | `ss -tlnp | grep :22` | 建议非标准端口 |
| SSH 配置 | `sshd -T` | 安全配置 |
| Web 服务 | `ss -tlnp | grep :80/:443` | HTTPS 强制 |
| 数据库端口 | `ss -tlnp | grep :3306/:5432` | 不暴露公网 |

### 2.3 网络防护

| 检查项 | 标准 |
|--------|------|
| fail2ban 安装 | 防止暴力破解 |
| DDoS 防护 | 云厂商/CDN 防护 |
| 端口扫描检测 | 有监控 |
| 异常流量告警 | 有配置 |

## 三、应用安全（Application Security）

### 3.1 Web 服务

| 检查项 | 标准 |
|--------|------|
| HTTPS 证书 | 有效且自动续期 |
| HTTP 重定向 | 强制跳转 HTTPS |
| 安全头 | HSTS、CSP、X-Frame-Options |
| 目录遍历 | 已禁用 |
| 错误信息 | 不泄露敏感信息 |

### 3.2 数据库

| 检查项 | 标准 |
|--------|------|
| 版本 | 最新稳定版 |
| 认证 | 强密码/密钥 |
| 授权 | 最小权限 |
| 网络 | 仅内网访问 |
| 备份 | 定期自动备份 |
| 加密 | 敏感数据加密存储 |

### 3.3 容器安全（Docker）

| 检查项 | 命令 | 标准 |
|--------|------|------|
| Docker 版本 | `docker version` | 最新稳定版 |
| 容器用户 | `docker inspect` | 非 root |
| 镜像扫描 | `docker scan` | 无高危漏洞 |
| 网络隔离 | `docker network ls` | 有隔离 |
| 资源限制 | `docker stats` | CPU/内存限制 |

### 3.4 密钥与凭证

| 检查项 | 标准 |
|--------|------|
| SSH 密钥 | 使用 Ed25519，禁用密码 |
| API 密钥 | 环境变量/密钥管理 |
| 数据库密码 | 不硬编码 |
| Git 凭证 | 不使用明文密码 |
